Belirsiz sayýda parametre alan ve bu parametrelerin standart sapmasýný veren stdsap() isimli bir fonksiyon yazalým. Standart sapmanýn tanýmlanabilmesi için fonksiyon en az iki parametre almalý. Bunun dýþýnda parametre sayýsýnda bir kýsýtlama olmayacak. Bu þartý saðlamak için fonksiyon arayüzünü stdsap(x1, x2, *y) þeklinde yazmamýz gerekir.




import math   # karekök için

def stsapma(x1, x2, *y):
    N = len(y) + 2 # Fonksiyona verilen parametre sayýsý

    # ortalamayý hesapla
    toplam = x1 + x2
    for z in y:
        toplam += z
    ort = toplam / N

    karetoplam = (x1-ort)**2 + (x2-ort)**2
    for z in y:
        karetoplam += (z-ort)**2
    stsap = math.sqrt(karetoplam / (N-1))

    return stsap
